# NutriSmart

NutriSmart é uma aplicação web moderna construída com Next.js, Firebase e IA generativa, projetada para ajudar os usuários a rastrear sua nutrição e alcançar seus objetivos de saúde.

## Funcionalidades Principais

- **Diário Nutricional:** Registre suas refeições e consumo de água de forma fácil e intuitiva.
- **Chef Virtual com IA:** Obtenha sugestões de receitas personalizadas com base nos seus ingredientes e preferências.
- **Análise de Desempenho:** Acompanhe seu progresso de calorias, macronutrientes e peso com gráficos detalhados.
- **Histórico Completo:** Revise seu consumo diário para identificar padrões e se manter no caminho certo.
- **Metas Personalizadas:** Defina e ajuste suas metas de calorias, proteínas e hidratação.

## Tecnologias Utilizadas

- **Framework:** Next.js (com App Router)
- **Linguagem:** TypeScript
- **UI:** React, ShadCN UI
- **Estilização:** Tailwind CSS
- **Backend & DB:** Firebase (Authentication e Firestore)
- **IA Generativa:** Genkit

Para começar a explorar o projeto, dê uma olhada no arquivo `src/app/page.tsx` para ver a landing page ou em `src/app/dashboard/page.tsx` para a tela principal da aplicação.
